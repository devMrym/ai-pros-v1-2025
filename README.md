# Bootcamp Labs and Final Project

This repository contains the labs and assignments completed during the **4th week of the bootcamp**, as well as the **final project** focused on **horse breed classification**.

## Project Overview

The **Horse Classification** project includes a classificatiom model to classify images of horses into 7 different breeds:

- Akhal-Teke
- Appaloosa
- Orlov Trotter
- Vladimir Heavy Draft
- Percheron
- Arabian
- Friesian

The project demonstrates key deep learning concepts, including:

- Transfer learning with pre-trained models (ResNet18)
- Data preprocessing and augmentation
- Model training, validation, and evaluation
- Visualization of predictions
